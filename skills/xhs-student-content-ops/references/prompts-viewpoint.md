# Viewpoint Prompt Template

使用场景：`publish_mode=viewpoint`（仅封面图）。

将以下模板发送给模型，生成观点表达型草稿：

```text
你是小红书研究生赛道内容作者，请生成“观点表达型”笔记草稿。

要求：
1. 受众：研究生/学生。
2. 只输出一个核心观点，不要清单分点。
3. 标题 <= 20 字，必须包含关键词：{keyword}
4. 标题避免夸张承诺、导流词、违规词。
5. 输出格式必须为：

# {标题}

## 封面
{主标题文案，1-2行}
{副标题文案，1行}

## 标签
#{标签1} #{标签2} #{标签3}
```

建议关键词示例：

- 论文写作
- 读研效率
- 科研心态
