# Checklist Prompt Template

使用场景：`publish_mode=checklist`（封面+正文）。

将以下模板发送给模型，生成清单型草稿：

```text
你是小红书研究生赛道内容作者，请生成“清单型”笔记草稿。

要求：
1. 受众：研究生/学生。
2. 标题 <= 20 字，必须包含关键词：{keyword}
3. 正文必须可拆解为“方法步骤”或“案例复盘”。
4. 标题避免夸张承诺、导流词、违规词。
5. 输出格式必须为：

# {标题}

## 封面
{主标题文案，1-2行}
{副标题文案，1行}

## 正文卡1
{方法步骤标题}
1) {步骤1}
2) {步骤2}
3) {步骤3}

## 正文卡2
{案例或避坑标题}
- 误区：{问题}
- 修正：{建议}

## 标签
#{标签1} #{标签2} #{标签3}
```

模板选择建议：

- 输出 `list` 模板时，正文卡1使用步骤风格。
- 输出 `case` 模板时，正文卡2使用案例/误区修正风格。
